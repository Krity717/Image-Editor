# Image Editor (Python + CustomTkinter)

A lightweight desktop image editor built using Python, Pillow, and CustomTkinter.

## Features

- Open and save images
- Apply image filters
- Adjust brightness and contrast
- Rotate and flip images
- Undo and reset changes
- Modern dark UI

## Tech Stack

- Python 3.x
- CustomTkinter
- Pillow (PIL)
- Tkinter

### Install dependencies
